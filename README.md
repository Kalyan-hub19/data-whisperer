# 🧠 Data Whisperer

A conversational CSV analysis tool powered by **LLaMA 3**, **LangChain**, and **Streamlit**.  
Upload any CSV file and ask natural language questions to get instant answers, charts, or summaries — with session memory powered by **MongoDB**.

---

## 🚀 Features

- 📁 Upload your own CSV file
- 💬 Ask natural language questions about your data
- 🔁 Maintains chat context using MongoDB
- 🦙 Powered by Meta LLaMA 3 via Together AI
- ⚙️ Built with LangChain + Streamlit
- 🧠 Supports persistent memory (like ChatGPT)

---

## 🛠 Tech Stack

- **Frontend**: Streamlit
- **LLM**: Meta LLaMA 3 (via Together API)
- **Framework**: LangChain
- **Memory Backend**: MongoDB (Compass or Atlas)
- **Languages**: Python 3.12+
